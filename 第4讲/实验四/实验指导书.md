#### 理论回顾

自注意力的核心公式：

$$\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right) V$$

其中：
- Q (Query)：查询矩阵，代表"我想找什么信息"
- K (Key)：键矩阵，代表"我能提供什么索引"
- V (Value)：值矩阵，代表"我实际的信息内容"
- $d_k$：Key 向量的维度，用于缩放防止梯度过小

### 任务2：BERT 情感分类微调（65分）

#### 任务描述

基于 `hotel.csv` 酒店评论数据集，对 `bert-base-chinese` 模型进行情感分类微调，实现正面/负面评论的二分类。

#### 理论回顾

**预训练 + 微调范式**：
1. **预训练阶段**：BERT 在大规模中文语料上通过 MLM（遮盖语言模型）任务学习通用语言理解能力
2. **微调阶段**：在预训练好的 BERT 基础上，添加分类头（线性层），用带标签的下游任务数据训练

**微调训练循环**：
```
for each epoch:
    for each batch:
        1. 前向传播：input → model → logits → loss
        2. 梯度清零：optimizer.zero_grad()
        3. 反向传播：loss.backward()
        4. 参数更新：optimizer.step()
```

### 任务3（选做）：LoRA 轻量微调对比（+30分）

#### 任务描述

使用 `peft` 库对 BERT 进行 LoRA 微调，并与任务2的全参数微调进行对比。

#### 理论回顾

**LoRA 核心思想**：
- 全参数微调更新整个权重矩阵 W，参数量巨大
- LoRA 假设微调时的权重变化 ΔW 是低秩的：ΔW = AB
  - A: shape (d, r)，B: shape (r, d)，r << d
  - 只需训练 A 和 B，原始 W 冻结不动
- 类比 PCA：用低维空间近似高维变化
