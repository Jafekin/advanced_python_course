# 实验八：基于航天产业报告的智能问答 Agent

## 一、实验目的

1. 掌握 LangChain 框架中 Agent 的构建方法
2. 理解 ReAct（Reasoning + Acting）范式的推理-行动循环机制
3. 学会将 RAG 检索能力封装为 Agent 可调用的工具（Tool）
4. 掌握 Agent 多轮对话记忆管理（ChatMessageHistory）
5. 学会用 Gradio 构建可交互的智能问答界面
6. 综合运用第1-6讲所学知识，完成课程最终 AI 应用交付物

## 二、实验环境

- **平台**：CloudStudio + CodeBuddy
- **语言**：Python 3.10+
- **核心依赖**：

```bash
pip install langchain langchain-openai langchain-community \
    chromadb sentence-transformers gradio markitdown
```

- **LLM 服务**：Ollama 本地部署（第6-7讲已安装）；建议使用硅基流动免费模型，速度较快

**评分要点**：
- RAG 流水线完整性（加载→分块→向量化→存储→检索）（10分）

### 任务2：ReAct Agent 构建（25分）

**目标**：基于 LangChain 构建一个带 RAG 工具和计算器工具的 ReAct Agent。

**要求**：
1. 使用 `ChatOpenAI` 连接 Ollama 本地 LLM
2. 定义计算器工具（`@tool` 装饰器）
3. 使用 `create_react_agent` 创建 ReAct Agent
4. 使用 `AgentExecutor` 包装 Agent，设置 `verbose=True` 和 `max_iterations=5`
5. 编写 ReAct 格式的 Prompt 模板（包含 Thought/Action/Observation 格式说明）

**关键代码框架**：

```python
from langchain.agents import create_react_agent, AgentExecutor
from langchain_core.prompts import PromptTemplate

# 1. 初始化 LLM
llm = ChatOpenAI(model="qwen2.5:7b-instruct-q4_K_M", ...)

# 2. 定义计算器工具
@tool
def calculator(expression: str) -> str:
    """..."""
    pass

# 3. 工具列表
tools = [rag_search, calculator]

# 4. ReAct Prompt 模板
prompt = PromptTemplate.from_template("...")

# 5. 创建 Agent
agent = create_react_agent(llm, tools, prompt)
agent_executor = AgentExecutor(agent=agent, tools=tools, ...)
```

**测试用例**（至少完成以下3个）：
1. 纯检索问题："报告中提到了哪些航天企业？"
2. 需要计算的问题："如果发射次数每年增长10%，从70次开始，3年后是多少？"
3. 多步推理问题："2024年中国航天发射次数是多少？和2023年相比增长了多少？"

**评分要点**：
- Agent 构建完整性（LLM + 工具 + Prompt + Executor）（10分）
- ReAct Prompt 格式正确性（10分）
- 3个测试用例通过（5分）

### 任务3：多轮对话记忆（25分）

**目标**：为 Agent 添加对话历史管理，使其能够理解上下文追问。

**要求**：
1. 使用 `ChatMessageHistory` 管理消息历史
2. 使用 `RunnableWithMessageHistory` 包装 AgentExecutor
3. 实现 session 隔离（不同用户的对话互不影响）
4. 更新 Prompt 模板以包含 `{chat_history}` 占位符
5. 演示至少 3 轮连续对话（后续提问引用前面的内容）

**多轮对话测试场景**：

```
第1轮: "报告中提到了哪些航天企业？"

第3轮: "帮我计算一下这个企业的市值增长率"（引用第2轮结果）
```

**评分要点**：
- ChatMessageHistory 实现正确（5分）
- RunnableWithMessageHistory 包装正确（5分）
- Prompt 模板包含 chat_history（5分）
- 3轮连续对话演示（Agent 正确理解上下文）（10分）

### 任务5：Agent 行为分析（10分）

**目标**：记录并分析 Agent 的推理轨迹，理解 ReAct 范式的实际运行过程。

**要求**：
1. 选择 1 个多步推理问题，记录完整的 Thought-Action-Observation 轨迹
2. 绘制该问题的 Agent 决策流程图（可用文本、Markdown 或图片）
3. 分析：Agent 在哪些步骤做出了正确决策？是否有不必要的工具调用？

**报告格式**：

```

  Thought 1: ...

  Observation 1: ...
  Thought 2: ...
  ...

  - 正确决策：...
  - 不必要的步骤：...
```

**评分要点**：
- 完整记录推理轨迹（5分）

## 五、实验提示

1. **复用第6-7讲代码**：RAG 流水线的核心逻辑（分块、向量化、检索）直接复用，只需封装为 Tool
2. **调试技巧**：设置 `verbose=True` 可以看到 Agent 的完整推理过程
3. **常见问题**：
   - Agent 不调用工具 → 检查工具描述是否清楚
   - Agent 无限循环 → 确认设置了 `max_iterations`
   - 对话历史不生效 → 确认 Prompt 中有 `{chat_history}` 占位符
4. **安全提示**：`calculator` 工具使用 `eval()` 存在安全风险，仅限教学演示使用
