# 国际游客到访量：实验数据说明

## 1. 数据用途

本数据集用于《第1讲：Python 数据处理 × Vibe Coding》的实验部分，**与课堂案例使用的“航天进出口额.xlsx”完全不同**。实验主题是：比较不同国家国际游客到访量的规模、区域差异和 2020 年的变化。

## 2. 数据来源

- 数据提供方：Our World in Data（OWID）
- 原始指标：Arrivals of tourists from abroad
- 原始数据来源：UN Tourism
- 原始下载地址：`https://ourworldindata.org/grapher/international-tourist-arrivals.csv`
- 下载日期：2026-07-21

## 3. 本课程使用的子集

为保证课堂运行速度和分析任务的可控性，本文件从原始数据中筛选得到：

- 国家/地区：20 个
- 年份：2010—2020 年
- 记录数：213 行
- 文件格式：UTF-8 with BOM 编码 CSV
- 排序方式：先按年份，再按国家名称排序

2020 年保留在数据中，用于分析疫情冲击下国际旅游活动的变化；该年份并不代表完整的正常年度水平，分析结论应谨慎表述。

## 4. 字段说明

| 字段 | 含义 |
|---|---|
| 国家 | 国家或地区名称 |
| 国家代码 | 三位字母国家代码 |
| 年份 | 统计年份 |
| 国际游客到访量 | 来自国外的游客到访次数，不等同于独立游客人数 |
| 世界区域 | OWID 提供的世界区域分类 |

## 5. 清洗与使用约定

1. 使用 `pd.read_csv(..., encoding="utf-8-sig")` 读取。
2. `国际游客到访量` 应转换为数值类型。
3. 本教学子集没有人为填补缺失值；学生仍需检查缺失值，并说明缺失值处理策略。
4. 计算增长率时，第一年没有上一年数据，应产生缺失值，不能简单当作 0%。
5. 2020 年可能受到全球疫情影响，应将其作为特殊年份解释，不能直接与普通年份等价比较。
