## 二、Vibe Coding 操作方法

### 两段式 Prompt

- Round 1：描述你想要的"感觉"和大方向，不要报具体的 API 参数名
- Round 2：在保持方向的前提下，用逐条约束收紧细节

### Diff 审查三步

每次 AI 修改代码后，先看 Diff，确认以下三点再点 Accept：

1. 字段名是否和你的数据文件一致？
2. 筛选方向、排序方向、算法选择是否正确？
3. 空值、编码、图表元素有没有遗漏？

## 八、常见问题

**Q：航天进出口额.xlsx 的指标列怎么解析？**
A：用 `df['指标'].apply(lambda s: s.split('_'))` 拆分。指标列形如 `进口额(人民币)_(HS88章)航空器、航天器及其零件_日本_当期`，按 `_` split 后倒数第二段是国家，第一段含"进口额"或"出口额"。如果 AI 解析错了，把样例贴给它让它修。

**Q：月份列（2022-01 等）有缺失值怎么办？**
A：航天数据是真实业务数据，部分国家某些月份无贸易记录是正常的。两种策略：`dropna(subset=month_cols, how='any')` 删除缺失行，适合严格分析；`fillna(中位数)` 填充，适合保留样本量。告诉 AI 你的选择，让它说明理由。

**Q：24 个月份列名怎么批量选出来？**
A：用列表推导式 `month_cols = [c for c in df.columns if c.startswith('2022') or c.startswith('2023')]`，然后 `df[month_cols].sum(axis=1)` 按行求和得年度合计。

**Q：karate.gml 用 `nx.read_gml()` 读入后节点标签是字符串？**
A：告诉 AI："节点标签应该是整数，请修正"，不要自己查文档。

**Q：airports.json 有8.5MB，Python 会不会内存不够？**
A：约5万条记录，可以正常处理。如果报错，把错误信息直接发给 AI 让它修复。

**Q：3D 散点图看不清分布？**
A：告诉 AI："点调小（s=1），透明度提高（alpha=0.3），视角仰角30方位角45"。
