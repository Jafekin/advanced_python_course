# 实验1 实验指导书：国际游客到访量数据处理全流程

## 实验定位

本实验使用一份与课堂案例不同的公开 CSV 数据，课堂案例继续保留并使用 `第1讲/实验一/航天进出口额.xlsx`。本实验围绕国际旅游数据练习 Pandas 的文件读写、缺失值检查、分组聚合、时间序列变化和可视化。

## 实验目标

1. 掌握 Pandas 对 CSV 数据的加载、探索与基本信息获取。
2. 掌握字段类型检查、缺失值分析、重复值检测和异常年份识别。
3. 掌握按国家、区域和年份进行分组聚合、排序和排名。
4. 掌握 `shift()`、增长率、`pivot()`、`melt()` 等时间序列与数据重塑操作。
5. 全程运用 Vibe Coding 方法（两段式 Prompt + Diff 审查 + @引用），培养人机协作编程能力。

## 实验环境

- 平台：CloudStudio + CodeBuddy
- 语言：Python 3.x
- 核心库：pandas、numpy、matplotlib
- 数据集：`国际游客到访量.csv`
- 数据说明：`国际游客到访量_数据说明.md`
- 数据规模：213 行 × 5 列，20 个国家/地区，2010—2020 年

## 数据集说明

数据字段如下：

| 字段 | 含义 |
|---|---|
| 国家 | 国家或地区名称 |
| 国家代码 | 三位字母国家代码 |
| 年份 | 统计年份 |
| 国际游客到访量 | 来自国外的游客到访次数，不等同于独立游客人数 |
| 世界区域 | 世界区域分类 |

2020 年保留在数据中，用来讨论特殊年份冲击。分析时不能把 2020 年的下降简单解释为长期趋势。

## Vibe Coding 操作方法

### 两段式 Prompt

每次向 AI 提需求时，按“氛围段 + 约束段”组织：

```text
【氛围段】我正在分析不同国家的国际游客到访量，希望比较旅游规模和年度变化。

【约束段】请用 pandas 实现：1) 读取国际游客到访量.csv 2) 按国家汇总 2010—2019 年到访量 3) 取 Top10 4) 输出排序表。要求变量名用英文、注释用中文，并显式处理 UTF-8 编码。
```

### Diff 审查三步清单

1. **结构检查**：导入是否齐全？文件路径、编码和变量名是否一致？
2. **逻辑检查**：筛选条件是否符合题意？增长率分母是否为上一年？分组后是否重复计数？
3. **边界检查**：第一年增长率是否为缺失？缺失值、零值和 2020 年特殊年份是否被正确处理？

### @文件引用

在 CodeBuddy 对话中使用 `@国际游客到访量.csv` 和 `@国际游客到访量_数据说明.md`，让 AI 获取真实列名、字段含义与数据范围。

## 任务2：字段检查与数据清洗（30分）

### 任务描述

检查数据类型、缺失值和重复记录，构造可用于时间序列分析的增长率字段，并识别 2020 年这一特殊年份。

### 具体要求

1. 检查列名是否完整，确认 `国际游客到访量` 为数值类型。
2. 统计每列缺失值数量与占比。
3. 检查重复行，并说明是否需要删除。
4. 按国家和年份排序，使用 `groupby().shift(1)` 计算上一年到访量。
5. 计算年度增长率：

   ```python
   (本年到访量 - 上年到访量) / 上年到访量 * 100
   ```

6. 对第一年无上一年数据的情况保留缺失值，不能填成 0%。
7. 增加 `是否特殊年份` 字段，标记 2020 年。
8. 用 IQR 方法检测 `国际游客到访量` 的极端值，并说明“统计异常”不一定等于“业务错误”。
9. 记录 Diff 审查发现的问题。

### 示例 Prompt

```text
【氛围段】数据已经读入，但我需要确认它是否适合比较不同国家的年度旅游变化。

【约束段】请实现：1) 检查列名、数据类型、缺失值和重复行；2) 按国家、年份排序；3) 用 groupby + shift 计算上一年到访量；4) 计算增长率百分比；5) 标记 2020 年；6) 用 IQR 检测到访量异常值；7) 第一年的增长率保留 NaN。代码分步输出中间结果。
```

### 预期输出

- 数据质量报告。
- 包含 `上年到访量`、`增长率%`、`是否特殊年份` 的清洗后 DataFrame。
- 缺失值、重复行和 IQR 检测报告。
- Diff 审查记录（至少记录 1 个实质发现）。

### 评分标准

| 评分项 | 分值 | 标准 |

| 类型与缺失值检查 | 8分 | 检查完整，结论准确 |

## 任务3：分组聚合与深度统计（30分）

### 任务描述

基于清洗后的数据，分析国家排名、区域差异、年度趋势和 2020 年变化。

### 具体要求

1. **国家 Top10**：统计 2010—2019 年各国国际游客到访量总和，取 Top10 并绘制水平条形图。
2. **区域统计**：按世界区域分组，计算平均年度到访量和国家数量，排序输出。
3. **年度趋势**：统计所有国家每年的总到访量和同比增长率，绘制折线图。
4. **国家趋势**：选取 Top3 国家，使用 `pivot()` 或 `melt()` 绘制年度趋势折线图。
5. **2020 年对比**：比较各国 2019—2020 年到访量变化率，列出下降幅度最大的 5 个国家。
6. 至少制作两张有标题、坐标轴标签和图例的图表。

### 示例 Prompt

```text
【氛围段】我想比较不同国家的旅游规模，并观察 2020 年特殊年份对国际游客到访量的影响。

【约束段】请实现：1) 排除 2020 年后统计各国 2010—2019 年总到访量 Top10；2) 按世界区域计算平均到访量；3) 统计年度总量和同比增长率；4) 选 Top3 国家画趋势图；5) 计算 2019—2020 年变化率并输出降幅最大的5个国家；6) 图表使用中文标题和标签。
```

### 预期输出

- 国家 Top10 水平条形图。
- 区域聚合统计表。
- 年度总量趋势图。
- Top3 国家趋势图。
- 2019—2020 年下降幅度 Top5 表。

### 评分标准

| 评分项 | 分值 | 标准 |
|---|:---:|---|

## 任务4：AI 辅助综合分析（25分）

### 任务描述

### 推荐问题

> 2019 年各国国际游客到访量较高的国家，在 2020 年是否都受到相似程度的冲击？

也可以自拟问题，例如：

- 哪个世界区域的旅游恢复速度更快？
- 2010—2019 年哪些国家的增长最稳定？

### Vibe Coding 全流程

1. 至少 3 轮 Prompt 迭代（首次生成 → 发现问题 → 修正 → 优化）。
2. 每轮记录 Prompt 内容、AI 输出和你的审查意见。
3. 使用 `@国际游客到访量.csv` 和 `@国际游客到访量_数据说明.md` 提供上下文。
4. 按结构、逻辑、边界三类检查 Diff。
5. 最终结论必须引用表格或图表中的具体数值。

### 示例 Prompt 迭代过程

```text
【Round 1】
氛围段：我想了解 2020 年国际旅游受到的冲击。
约束段：计算每个国家 2019 年和 2020 年到访量的变化率并排序。

→ 审查发现：部分代码没有明确排除缺失值，也没有检查 2019 和 2020 是否同时存在。

【Round 2】
氛围段：上一版结果可能把缺失年份当成 0，影响变化率。
约束段：只保留同时具有 2019 和 2020 年数据的国家；用 merge 对齐年份；输出变化率和原始到访量。

→ 审查发现：逻辑正确，但还需要区分统计变化和疫情背景，避免把变化直接解释为长期趋势。

【Round 3】
氛围段：现在需要让结果更容易比较和解释。
约束段：绘制 2019—2020 年到访量变化率 Top10 条形图，添加零线、数值标签和注释，标题明确写出“特殊年份”，并在结论中说明相关限制。
```
