# 实验1 实验指导书：航天贸易数据处理全流程

## 实验目标

1. 掌握 Pandas 对 Excel 数据的加载、探索与基本信息获取
2. 掌握数据清洗核心操作：列解析、缺失值处理、异常值检测
3. 掌握分组聚合、趋势分析与相关性计算
4. 全程运用 Vibe Coding 方法（两段式 Prompt + Diff 审查 + @引用），培养人机协作编程能力

## 实验环境

- 平台：CloudStudio + CodeBuddy
- 语言：Python 3.x
- 核心库：pandas, numpy
- 数据集：`航天进出口额.xlsx`（76行 × 29列，42个国家/地区，2022-2023年共24个月度数据，含自然缺失值）

## Vibe Coding 操作方法

本实验要求全程使用 Vibe Coding 方式完成代码编写，以下是核心操作要点：

### 两段式 Prompt

每次向 AI 提需求时，按"氛围段 + 约束段"组织：

```text
【氛围段】我正在分析航天进出口贸易数据，需要找出进口额最大的国家并观察其月度变化趋势。

【约束段】请用 pandas 实现：1) 按国家分组求进口总额 2) 取 Top10 3) 绘制这10国的月度折线图。要求：变量名用英文、注释用中文、使用链式调用。
```

### Diff 审查三步清单

AI 生成代码后，按以下顺序审查：

1. **结构检查**：导入是否齐全？函数/变量命名是否一致？
2. **逻辑检查**：数据流是否正确？筛选条件是否遗漏？聚合方式是否符合需求？
3. **边界检查**：空值是否处理？类型是否匹配？索引是否对齐？

### @文件引用

在 CodeBuddy 对话中使用 `@航天进出口额.xlsx` 引用数据文件，让 AI 自动获取列名、数据类型等上下文信息，减少 Prompt 中的重复描述。

## 任务2：指标列解析与数据清洗（30分）

### 任务描述

原始数据的第一列为复合指标（如"进口_美国"），需要拆分为"方向"和"国家"两列，并对整个数据集进行缺失值分析与处理、异常值检测。

### 具体要求

1. **指标解析**：将第一列按分隔符拆分为"贸易方向"（进口/出口）和"国家/地区"两个新列
2. **缺失值分析**：
   - 统计每列缺失值数量与占比
   - 选择合适的填充策略（说明选择理由）
   - 实施填充并验证
3. **异常值检测**：
   - 选取任意 3 个月度数值列
   - 分别用 IQR 方法和 Z-score 方法检测异常值
   - 输出异常值检测报告（含异常值数量、所在行、具体值）
4. **Diff 审查**：对 AI 生成的清洗代码执行三步清单审查，记录发现的问题

### 示例 Prompt

```text
【氛围段】数据集的第一列是"贸易方向_国家"的复合格式，我需要拆开它们以便后续分组分析。同时数据有自然缺失值需要处理。

【约束段】请实现：1) 用 str.split('_', expand=True) 拆分第一列为"贸易方向"和"国家地区" 2) 统计所有列的缺失率 3) 对数值列用前向填充(ffill)，对分类列用众数填充 4) 验证填充后无缺失。代码分步执行，每步打印中间结果。
```

### 预期输出

- 拆分后的 DataFrame 含"贸易方向""国家地区"两个新列
- 缺失值统计表（列名、缺失数、缺失率）
- 填充前后对比验证
- 异常值检测报告（3列 × 2种方法）
- Diff 审查记录（至少记录 1 个发现）

### 评分标准

| 评分项 | 分值 | 标准 |
|--------|:---:|------|
| 列拆分正确 | 8分 | 正确解析为两列，无丢失 |
| 缺失值策略合理 | 8分 | 策略选择有理由，实施正确，验证通过 |
| 异常值检测完整 | 8分 | IQR 和 Z-score 两种方法均实现，报告清晰 |
| Diff 审查记录 | 6分 | 按三步清单执行，记录有实质发现 |

## 任务3：分组聚合与深度统计（30分）

### 任务描述

基于清洗后的数据，进行分组聚合、趋势分析、波动性排名与相关性计算。

### 具体要求

1. **进口 Top10**：按国家分组计算进口总额，取前10名，用条形图可视化
2. **月度趋势分析**：选取 Top3 进口国，绘制 24 个月的进口额折线图，标注趋势方向
3. **波动性排名**：计算每个国家进口额的标准差（或变异系数），排名前5的国家
4. **相关性分析**：计算 Top10 国家之间进口额的相关系数矩阵，找出高相关（>0.7）的国家对

### 示例 Prompt

```text
【氛围段】我已经完成了数据清洗，现在想深入分析各国航天贸易的规模和变化规律。

【约束段】请实现以下分析：
1) 筛选"进口"数据，按国家 groupby 后 sum 所有月度列，sort_values 取 Top10
2) 取 Top3 国家的24个月数据，melt 为长格式后画折线图
3) 计算每国进口额的 std 和 cv（变异系数），nlargest(5)
4) Top10 国家的月度进口额做 corr()，输出 >0.7 的国家对
要求：每步独立 cell，图表加中文标题和标签。
```

### 预期输出

- Top10 进口国条形图
- Top3 国家月度趋势折线图
- 波动性 Top5 排名表
- 相关系数矩阵与高相关国家对列表

### 评分标准

| 评分项 | 分值 | 标准 |
|--------|:---:|------|
| 分组聚合与 Top10 | 8分 | groupby/sum/sort 正确，图表清晰 |
| 趋势分析 | 8分 | 折线图正确反映 24 个月变化，有趋势说明 |
| 波动性计算 | 7分 | std 或 cv 计算正确，排名合理 |
| 相关性分析 | 7分 | corr 矩阵正确，高相关对识别准确 |

## 任务4：AI 辅助综合分析（25分）

### 任务描述

自选一个有价值的分析问题，全程使用 Vibe Coding 方式完成，重点考察 Prompt 迭代能力和 Diff 审查质量。

### 具体要求

1. **自选问题**（以下任选其一或自拟）：
   - 哪些国家的进出口存在显著不对称（贸易逆差/顺差）？
   - 2022 vs 2023 年度对比：哪些国家增长最快/下降最多？
   - 是否存在季节性波动？用 rolling 平滑后趋势如何？
2. **Vibe Coding 全流程**：
   - 至少 3 轮 Prompt 迭代（首次生成 → 发现问题 → 修正 → 优化）
   - 每轮记录：Prompt 内容 + AI 输出 + 你的审查意见
   - 使用 @文件引用提供数据上下文
3. **提交内容**：
   - 3 轮 Prompt 截图
   - Diff 审查记录（每轮标注结构/逻辑/边界三类检查结果）
   - 最终分析结果与结论

### 示例 Prompt 迭代过程

```text
【Round 1】
氛围段：我想看看各国航天贸易是否平衡。
约束段：计算每个国家的进口总额和出口总额，求差值，排序输出。

→ AI 输出代码 → 审查发现：没处理缺失值导致求和结果偏小

【Round 2】
氛围段：上一版没处理缺失值，结果不准。
约束段：在求和前先用 fillna(0) 处理缺失值，然后重新计算进出口差额。增加一列"逆差/顺差"标签。

→ AI 输出代码 → 审查发现：逻辑正确，但缺少可视化

【Round 3】
氛围段：数值分析完成了，需要直观展示。
约束段：对 Top10 贸易逆差和 Top10 贸易顺差国家画水平条形图，红色表示逆差，蓝色表示顺差，加标题和数值标注。
```

### 评分标准

| 评分项 | 分值 | 标准 |

| Prompt 迭代质量 | 8分 | 3轮迭代有递进逻辑，每轮改进有明确目标 |
