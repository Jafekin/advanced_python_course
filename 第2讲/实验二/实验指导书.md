## 实验目的

1. 综合运用 Matplotlib 和 Seaborn 对真实经济时间序列进行多维度可视化分析。
2. 掌握科研论文级图表的完整制作流程，包括排版规范、统计标注和配色合规。
3. 理解月度时间序列中的长期趋势、季节性和特殊时期变化。
4. 体验 AI 辅助可视化的三轮迭代工作流，培养 Prompt 设计与批判审查能力。

## 实验环境

- 平台：CloudStudio + CodeBuddy
- Python 3.x + matplotlib + seaborn + numpy + pandas + scipy
- AI 工具：CodeBuddy（Vibe Coding 模式）

## 数据集说明

本实验使用 `美国零售销售额.csv`，由 FRED 序列 `RSXFS` 整理得到。它是 2010 年 1 月至 2024 年 12 月的月度数据，共 180 条记录，单位为百万美元。

| 列名 | 说明 | 示例 |
|------|------|------|
| 日期 | 月度观测日期 | 2010-01-01 |
| 年份 | 观测年份 | 2010 |
| 月份 | 观测月份，1—12 | 1 |
| 零售销售额_百万美元 | 当月零售销售额 | 302325 |
| 年月 | 用于绘图和排序的年月字段 | 2010-01 |

数据来源与字段解释详见 `美国零售销售额_数据说明.md`。原始下载文件 `零售销售额_原始.csv` 也保留在实验二目录中，便于追溯数据处理过程。

> **解释边界**：该序列是名义金额，未进行通货膨胀调整。本实验重点是数据整理、可视化和统计表达，不应将金额增长直接解释为实际消费量增长。

### 任务1：Matplotlib 基础可视化（20分）

**目标**：使用 Matplotlib 观察月度销售额的时间趋势、年度差异和月度季节性。

#### 1.1 月度销售额趋势图（8分）

- 绘制 2010—2024 年月度零售销售额折线图。
- 使用 `rolling(window=12)` 计算并叠加 12 个月移动平均线。
- 原始月度曲线使用较细、半透明线条，移动平均线使用醒目颜色和较粗线条。
- 标注 2020 年 3—4 月附近的特殊变化，并在图中说明这只是时间上的变化标记，不等于因果证明。
- 标题建议为“美国月度零售销售额趋势（2010—2024）”，轴标签分别为“日期”和“百万美元”。

#### 1.2 各年度平均销售额柱状图（6分）

- 按 `年份` 分组计算年度平均零售销售额。
- 绘制年度平均销售额柱状图，并在柱顶标注数值（可使用“十亿美元”作为显示单位）。
- 突出显示 2020—2024 年或使用渐变色表达时间顺序。
- 标题建议为“各年度平均零售销售额”。

#### 1.3 月份季节性均值图（6分）

- 按 `月份` 计算 2010—2024 年的月度平均销售额。
- 使用折线图或 1×2 / 2×1 子图展示 12 个月的季节性模式。
- 横轴必须按 1 月到 12 月排序，不得按字符串默认顺序排序。
- 添加月份标签、数据点或数值标注，并说明可能的季节性高低点。

**通用要求**：

- 所有图表保存为 PNG，`dpi=300`。
- 使用 `plt.rcParams` 设置中文字体（如 `Arial Unicode MS`、`SimHei` 或系统可用字体）。
- 图片保存至 `实验二/output/` 目录。
- 图表应包含标题、轴标签、图例（若需要）和必要的数据来源说明。

### 任务2：Seaborn 高阶探索（25分）

**目标**：使用 Seaborn 的高阶统计图表分析月份、时期与销售额分布之间的关系。

#### 2.1 热力图：月份 × 年份（10分）

- 构造行=`月份`、列=`年份`、值=`零售销售额_百万美元` 的透视表。
- 使用 `sns.heatmap` 绘制热力图，并添加数值标注（建议换算为十亿美元后保留 1 位小数）。
- 使用连续型色图（如 `YlOrRd` 或 `viridis`），并为纵轴设置“1月”至“12月”的标签。
- 讨论长期变化与年内高低月份是否同时存在。

#### 2.2 分时期小提琴图（8分）

- 将数据分为三个时期：2010—2014、2015—2019、2020—2024。
- 使用 `sns.violinplot` 比较三个时期的月度销售额分布。
- 叠加箱线图或均值点，帮助读者识别中位数、离散程度和异常月份。
- 说明小提琴图中的分布变化不能单独证明某一经济因素造成了变化。

#### 2.3 FacetGrid 月份季节性分面图（7分）

- 对三个时期分别计算 1—12 月的平均销售额。
- 使用 `sns.FacetGrid` 按时期分面，每个分面绘制月份均值折线和数据点。
- 三个分面共享 Y 轴范围，以便比较时期之间的季节性水平。
- 在图注或报告中回答：2020—2024 年的季节性形状是否与前两个时期一致？

**通用要求**：

- 保存至 `实验二/output/` 目录，`dpi=300`。
- 适当调整 `figsize`、字体和旋转角度，保证热力图标注可读。
- 解释图表时同时关注水平变化、波动程度和季节性形状。

### 任务3：科研论文级图表（30分）

**目标**：制作一张适合论文或研究报告使用的单栏图，完整呈现时间序列、平滑趋势、不确定性和统计比较。

#### 3.1 主图绘制（15分）

- 绘制月度零售销售额原始曲线。
- 叠加 12 个月移动平均线。
- 基于 12 个月滚动标准差和有效样本数，计算移动平均的近似 95% 置信区间，并用 `fill_between` 绘制。
- 标注数据中的最高销售额月份和最高年度平均值年份。
- 使用英文标题、轴标签和图例，避免中文字体影响论文排版。

#### 3.2 统计标注（8分）

- 将年度平均销售额分为两个时期：2010—2019 和 2020—2024。
- 使用 `scipy.stats.ttest_ind(..., equal_var=False)` 进行 Welch 独立样本 t 检验。
- 在报告中填写两个时期的样本量、均值、标准差、t 统计量、p 值和显著性判断。
- 在主图中标出两个时期的均值水平或在图注中报告统计结果。
- 报告中应注明：时期样本量不同，且年度观测不是完全随机样本，t 检验结果应作为描述性比较而非因果结论。

#### 3.3 发表级排版要求（7分）

| 检查项 | 要求 |
|--------|------|
| 图片宽度 | 3.5 inch 单栏宽度 |
| 分辨率 | PNG 300 dpi，同时输出 PDF |
| 背景 | 白底、无不必要网格 |
| 字体 | Arial 或系统可用无衬线字体 |
| 轴标签字号 | 8 pt 左右 |
| 配色 | Wong 2011 色盲友好配色 |
| 坐标轴 | 保留左、下边框，隐藏右、上边框 |
| 文件名 | `fig_nature.png` 与 `fig_nature.pdf` |

### 任务4：AI 辅助三轮迭代（25分）

选择任务3的科研主图，记录从需求描述到局部修订的过程。每一轮都要保存代码版本或截图，并人工检查 AI 输出。

#### Round 1 — 氛围提示

示例 Prompt：

> 请用 `美国零售销售额.csv` 画一张有科研感的月度零售销售额趋势图，突出长期变化和季节性，使用清晰的折线图。

要求：记录 AI 生成结果的优点和不足，例如只有原始曲线、缺少不确定性表达、字号和输出规格不明确等。

#### Round 2 — 约束提示

在 Round 1 基础上补充明确约束：

1. 增加 12 个月移动平均线和近似 95% 置信区间带。
2. 标注最高销售额月份和最高年度平均值年份。
3. 对 2010—2019 与 2020—2024 年度均值做 Welch t 检验并报告 p 值。
4. 使用 Wong 2011 色盲友好配色。
5. 设置 3.5 inch 宽、300 dpi、约 8 pt 字号，输出 PNG + PDF。
6. 隐藏顶部和右侧边框，避免不必要网格。

#### Round 3 — Inline Edit 微调

至少完成两处局部微调，并记录 Inline Edit Prompt、修改内容和修改后的效果，例如：

- 调整最高点箭头位置，避免遮挡原始曲线。
- 将置信区间透明度从 0.15 调整为 0.12。
- 将日期刻度改为每两年显示一次，减少拥挤。
- 调整图例位置和注释字号。

#### 4.3 对比分析（5分）

将 Round 1 和 Round 3 并排比较，从以下维度分析改进：

- 信息传达清晰度
- 视觉美观度
- 科研规范符合度
- 代码简洁度
