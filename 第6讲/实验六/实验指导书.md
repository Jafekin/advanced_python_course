## 任务3：向量化与存储（25分）

### 目标

使用 sentence-transformers 将分块向量化，存入 ChromaDB。

### 具体要求

1. **加载嵌入模型**：

```python
from sentence_transformers import SentenceTransformer

embed_model = SentenceTransformer('all-MiniLM-L6-v2')
```

2. **批量向量化**：

```python
chunks = chunk_recursive(text, chunk_size=500, overlap=50)
embeddings = embed_model.encode(chunks, show_progress_bar=True)
print(f"向量维度: {embeddings.shape}")  # (N, 384)
```

3. **存入 ChromaDB**：

```python
import chromadb

client = chromadb.PersistentClient(path="./chroma_db")
collection = client.get_or_create_collection("paper_qa")

collection.add(
    ids=[f"chunk_{i}" for i in range(len(chunks))],
    documents=chunks,
    embeddings=embeddings.tolist(),
    metadatas=[{"chunk_id": i, "source": "综述论文.pdf"} for i in range(len(chunks))]
)
print(f"入库完成: {collection.count()} 条")
```

4. **验证检索**：用一个简单查询测试语义检索是否工作

```python
query = "What are the challenges of deploying LLMs on mobile devices?"
query_vec = embed_model.encode(query)
results = collection.query(query_embeddings=[query_vec.tolist()], n_results=3)
for i, doc in enumerate(results['documents'][0]):
    print(f"[{i+1}] {doc[:100]}...")
```
